1.     Equipment

1.1 问题1 EQP00001

关键字

Equipment，Job Count

描述

设备Job Count报错，实际上设备中没有基板;

原因

设备的基板发生破片，被人为取走，导致设备中还残留Job Data，Job Count报错；

解决方案

设备手动将此Job Data做Delete动作，从而reset Job Count；

 

 

 

 

 

 

 

2.     Port & Cassette

2.1 问题1 CST00001

关键字

Port，Cassette

描述

Cassette无法正常搬入Port;

原因

Port Status 或者 Port Transfer Type Error；

解决方案

1.  确认Port Status为load request;

2.  确认Port Transfer Type为Stocker Inline；

 

 

 

2.2 问题2 CST00002

关键字

Cassette, Port

描述

Online Mode ,Cassette Load一直被Cancel;

原因

MES在下资料的时候,可能是EQP ID/Port ID/Cassette Sequence Number/Glass number and Glass位置与机台上的不符；

解决方案

请MES确认EQP ID/Port ID/Cassette Sequence Number/Glass number and Glass 位置资料是否正确

 

 

2.3 问题3 CST00003

关键字

Cassette, Port

描述

Cassette Load一直被Cancel;

原因

BC在下 资料的时候,可能是资料不对，导致被Cancel

解决方案

请BC确认在资料是否正确

 

 

2.4 问题4 CST00004

关键字

Cassette, Port

描述

Cassette Load一直被Cancel;

原因

BC在下 资料的时候,机台Check Glass 信息时,不符合机台的信息，导致被Cancel

解决方案

请机台确认机台check的Glass与BC下的资料存在的差异

 

2.5 问题5 CST00005

关键字

Cassette

描述

上Port成功后，Port的状态一直处于Waiting For Cassette的状态;

原因

当BC下start命令后，Robot一直未下Common命令

解决方案

1,当Robot是CPC控制时,请联系BC厂商

2,当Robot是ITC控制时,请联系Indexer厂商

 

2.6 问题6 CST00006

关键字

Cassette

描述

Port为Both Port时,当全部Glass做完 制程后回Cassette，Port没有自动的Unload;

原因

回Cassette的Glass里面的Check Data是否有为空值;

 

解决方案

请 确定Check Data是否有为空值.

 

2.7 问题7 CST00007

关键字

Port，Cassette

描述

无法从Port上搬走Cassette;

原因

Port Status 或者 Port Transfer Type Error；

解决方案

1.确认Port Status为load request;

2.确认Port Transfer Type为Stocker Inline；

 

 

 

2.8 问题8 CST00008

关键字

Port，Cassette

描述

Online情况下,无法正常上Port上;

原因

MES down的Chamberrunmode 为PLN, 而BC的是PL,两个资料不一致，导致无法正常上Port；

解决方案

请MES确认 down Chamberrunmode 资料是什么;

 

 

2.9 问题9 CST00009

关键字

Cassette

描述

Error:Cassette ID is empty

原因

此异常是因为设备漏报了Cassette ID；

解决方案

请确认有无输入Cassette ID或者联系设备厂商;

 

 

2.10   问题10 CST00010

关键字

Cassette Port

描述

在流片时，Glass 还没有回Cassette，BC就下了Process End 命令,导致机台内还存有Glass，无法继续流片

原因

解决方案

请厂商把Port Type的状态改成BothPort;

 

 

3.     Robot

3.1 问题1 R000000001

关键字

Robot, R000000001

描述

Robot没有收到BC指令动作，出现Error Code.

原因

Robot Status is (RUNNING) is not (IDLE) cannot issue Robot Command!

解决方案

1.确定Robot有没有在实体动作.

2.确定BC下给Robot的上一个指令是否完成.

3.2 问题 2 R000000003

关键字

Robot, R000000003

描述

Robot没有找到满足条件的设备，导致不能下命令出现Error Code.

原因

Robot cannot Find EQP Entity by Robot Node No (L2) and cannot issue Robot Command!

解决方案

确认设备是否有发Link Signal信号

3.3 问题3 R000000004

关键字

Robot, R000000004

描述

Indexer中的CIM Mode 是off状态,导致Robot不能下命令

原因

Robot Node No (L2) CIM Mode is (OFF) and cannot issue Robot Command!

解决方案

确认Indexer设备端的CIM Mode状态是否是CIM ON.

3.4 问题4 R000000005

关键字

Robot, R000000005

描述

BC下给Robot的命令之后,Bit没有正常的Off,导致不能下Robot命令

原因

Robot Control Command Reply Bit is (ON) and cannot issue Robot Command!

解决方案

请Robot厂商将此信号OFF.

3.5 问题5 R000000006

关键字

Robot, R000000006

描述

Robot没有找到可以控制的玻璃

原因

Robot cannot get any can control Job!

解决方案

请Robot厂商将此信号OFF.

3.6 问题6 R000000007

关键字

Robot, R000000007

描述

robot有命令[NO_COMMAND_ON_ROBOT_FAIL],导致下命令失败[只有在没有命令的状态下才能下命令,若indexer接收了命令，我们不会另外下命令]

原因

Robot has Command Status is (HAS_COMMAND_ON_ROBOT) is not (NO_COMMAND_ON_ROBOT)

Cannot issue Robot Command!

解决方案

请Robot厂商确认Robot现在是否有在执行命令.

3.7 问题7 R000000008

关键字

Robot, R000000008

描述

设备操作模式是MANUAL，导致不能下robot命令

原因

Robot Node No (L2) Operation Mode is (MANUAL) and cannot issue Robot Command!

解决方案

请厂商将MANUAL模式切换到AUTO模式

3.8 问题8 R000000009

关键字

Robot, R000000009

描述

设备操作模式是MANUAL，导致不能下robot命令

原因

Robot Node No (L2) Operation Mode is (MANUAL) and cannot issue Robot Command!

解决方案

请厂商将MANUAL模式切换到AUTO模式

6.     Common

6.1 问题1 COM00001

关键字

Recipe

描述

为什么Add Recipe之后，在界面上没有新增的那组Recipe

原因

解决方案

6.2 问题2 COM00002

关键字

Robot

描述

上Port完成，下栈之后，Robot不去取片

原因

Robot的状态是否自动& 机台是否发要片需求&设备的状态是否正确&机台内的Job Count 显示有值

解决方案

1.  请确认Robot的状态是否是Auto

2.  请确认机台是否有发要片需求

3.  请确认设备端的状态是否正确

4.  请确认机台端的Job Count 是否有值

 

 

6.3 问题3 COM00003

关键字

Recipe Check

描述

上Port成功后，下栈时Recipe check Reply NG.

原因

机台端Recipe Auto change的功能没有开启，导致recipe filter check NG

解决方案

请厂商把recipe auto change切成enable
